# ⚡ Household Power Consumption Analysis

A Python-based data analysis project that explores household electricity consumption patterns using historical power usage data. The project focuses on data preprocessing, exploratory data analysis (EDA), and visualization to uncover energy consumption trends, peak demand periods, and household usage behavior.

## 📌 Project Overview

This project analyzes household power consumption data to gain meaningful insights into electricity usage over time. By cleaning and transforming the dataset, the analysis highlights daily, weekly, and monthly consumption patterns through informative visualizations.

The project demonstrates essential data analysis techniques such as data cleaning, feature engineering, statistical analysis, and visualization using Python libraries.

---

## 🎯 Objectives

- Clean and preprocess raw household power consumption data.
- Perform Exploratory Data Analysis (EDA).
- Visualize electricity consumption trends.
- Identify peak power usage periods.
- Analyze relationships between voltage, current, and power consumption.
- Generate insights for improving energy efficiency.

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/yourusername/Household-Power-Consumption.git
```

2. Navigate to the project directory

```bash
cd Household-Power-Consumption
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Launch Jupyter Notebook

```bash
jupyter notebook
```

5. Open the notebook and run all cells.
